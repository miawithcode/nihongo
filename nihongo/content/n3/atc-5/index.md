## 「被动」、「使役」和「使役被动」的用法
|  | 表达 | 备注 |
| --- | --- | --- |
| **被动** | 受到他人行为的作用，或某个事项的影响 | **人**(和说话人心理亲近的人)作主语、**无生命体**作主语 |
| **使役** | **强制**他人做、**允许**他人做、**诱发**他人产生某事情感 |  |
| **使役被动** | **被迫**做某事、或**情绪不由自主**地受到外界影响而做某事 ||
